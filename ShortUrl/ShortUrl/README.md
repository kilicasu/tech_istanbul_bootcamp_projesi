# ShortURL Project

ShortURL, uzun URL’leri kısa ve kolay paylaşılabilir linklere dönüştüren basit bir web uygulamasıdır. Flask ve SQLite kullanılarak geliştirilmiştir.

## Özellikler

- Uzun URL’leri kısa URL’e dönüştürme
- Kısa URL’ler tıklanabilir ve doğrudan yönlendirme yapar
- Kayıtlı URL’leri listeleme
- Basit ve kullanıcı dostu arayüz
- Hatalı URL girişlerinde kullanıcıyı bilgilendirme

## Teknolojiler

- Python 3.x
- Flask
- SQLite
- HTML / CSS / Bootstrap
- Jinja2 (Template Engine)
